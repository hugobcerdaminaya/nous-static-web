# Ejemplo: blog simple

## Pagina

Una página describe como hacer una pagina web. Un  pagina tiene todo lo necesita:

  - los plantillas
  - los schemas

Y datos por defecto:
  - contenido textual
  - contenido visual

Para construir una página web con los datos por defecto:

```
nous pagina RUTA_POR_PAGINA_REPISOTORIO
```
